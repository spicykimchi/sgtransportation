Introduction

Assignment for Developing Data Products Course Project: Shiny Application and Reproducible Pitch. The application should use shiny and allow user to enter data. The chart or data should show the filtered data according to user's inpput.

Project

Analyse the transportation system in singapore for the past 10 years.

  1. Data Source

  2. https://data.gov.sg/dataset/public-transport-utilisation-average-public-transport-ridership

  3. Filtering Criteria

  4. Transport Type - Dropdown

  5. Year - Slider

  6. Application URL

  7. https://spicykimchi.shinyapps.io/DSMod9Wk1PJ/

Code

  1. Plot Chart

  2. Use renderPlot, plot and lines() to plot a line chart for each transport type

  3. Show Data in table

  4. Use sqldf to select data according to the relevant input passed to the shiny server

  5. Use renderTable
